# Quick Start Guide

Get up and running with the Copy Selected Tabs URLs extension in 2 minutes!

## üì¶ Installation (30 seconds)

1. Download or clone this repository
2. Open Chrome and type `chrome://extensions/` in the address bar
3. Toggle "Developer mode" ON (top-right corner)
4. Click "Load unpacked"
5. Select the folder containing this extension
6. Done! The extension icon appears in your toolbar

## üöÄ First Use (30 seconds)

### Try it now with 3 tabs:

1. **Open 3 tabs** - Open Google, GitHub, and Stack Overflow (or any sites)

2. **Select them all:**
   - Click the first tab
   - Hold **Shift** 
   - Click the third tab
   - All 3 tabs should now be highlighted!

3. **Copy the URLs:**
   - Click the extension icon in your toolbar
   - See all 3 tab titles and URLs displayed
   - Click **"Copy URLs to Clipboard"**
   - Success! ‚úì

4. **Paste anywhere:**
   - Open Notepad, Word, Slack, email, etc.
   - Press Ctrl+V (Windows) or Cmd+V (Mac)
   - You'll see all 3 URLs, one per line!

## üí° Pro Tips

### Select Specific Tabs (Not in a Row)
- Hold **Ctrl** (Windows/Linux) or **Cmd** (Mac)
- Click individual tabs you want
- Click extension icon and copy!

### Select All Tabs
- Click first tab
- Hold **Shift**
- Click last tab
- All tabs selected!

### Check Selected Tabs
The extension popup shows:
- How many tabs are selected
- Each tab's title and URL
- Real-time preview before copying

## üéØ Common Use Cases

- **Research:** Save all research tabs for later

- **Backup:** Save tab sessions before closing

## ‚ùì Troubleshooting

**Extension not appearing?**
- Make sure Developer mode is enabled
- Check that all files are in the folder
- Try refreshing the extensions page

**Can't select multiple tabs?**
- Make sure you're holding Shift or Ctrl/Cmd while clicking
- Selected tabs should be visually highlighted in Chrome

**Copy not working?**
- Click the extension icon (popup must be open to access clipboard)
- Check browser console for errors (right-click extension icon ‚Üí Inspect popup)
